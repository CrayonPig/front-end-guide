# 文件命名

市面上常用的命名规范：

  • camelCase（小驼峰式命名法 —— 首字母小写）
  • PascalCase（大驼峰式命名法 —— 首字母大写）
  • kebab-case（短横线连接式）
  • Snake（下划线连接式）

# HTML 文件名

全部采用小写方式， 优先选择单个单词命名，多个单词命名以下划线分隔。

|- error_report.html
|- success_report.html
